# 📖 About

**Ombak Biru** merupakan website pemesanan tiket kapal laut yang dikembangkan sebagai proyek mata kuliah **Web Programming**. Website ini bertujuan untuk memberikan kemudahan kepada pengguna dalam melakukan pemesanan tiket kapal secara online, mulai dari pencarian jadwal, pengisian data penumpang, proses pembayaran, hingga memperoleh **Electronic Boarding Pass (E-Ticket)**.

Dengan antarmuka yang modern dan responsif, Ombak Biru memberikan pengalaman pemesanan yang sederhana, cepat, dan efisien.

---

# ✨ Features

- 🏠 Landing Page Modern
- 🔍 Pencarian Jadwal Kapal
- 🚢 Detail Jadwal Perjalanan

# 🚀 Installation

### 1. Clone Repository

```bash
git clone https://github.com/Jahiatus/prjct-bsi
```

### 2. Masuk ke Folder Project

```bash
cd prjct-bsi
```

### 3. Pindahkan Project ke

```text
xampp/htdocs/
```

### 4. Jalankan

- Apache
- MySQL

### 5. Import Database

Import file

```text
database/dbwp.sql
```

ke phpMyAdmin.

### 6. Jalankan Website

```text
http://localhost/prjct-bsi
```

# 📸 Main Features

## 🏠 Homepage

Landing page modern yang menyediakan informasi layanan serta fitur pencarian jadwal kapal.

---

## 🔍 Schedule Search

Pengguna dapat mencari jadwal kapal berdasarkan:

- Pelabuhan Asal
- Pelabuhan Tujuan
- Tanggal Keberangkatan
- Kelas Kapal

---

## 👤 Booking

Mengisi data pemesan dan data penumpang secara lengkap sebelum melakukan pembayaran.

---

## ✅ Verification

Menampilkan ringkasan data pemesanan agar pengguna dapat memastikan seluruh informasi telah benar.

---

## 💳 Payment

Konfirmasi pembayaran menggunakan metode QRIS sebelum tiket diterbitkan.

---

## 🎫 Electronic Boarding Pass

Setelah pembayaran berhasil, pengguna akan memperoleh **Electronic Boarding Pass** yang berisi:

- Booking Code
- Detail Perjalanan
- Data Pemesan
- Data Penumpang
- QR Code
- Status Pembayaran
- Download PDF
